一个窗口，
装下所有文件。

HorseMD 是一款温暖安静的 Typora 风格 Markdown 编辑器 —— 所见即所得的书写画布， 加上 Typora 一直缺席的东西：标签页与文件树工作区。 每个文件都在同一个窗口里打开，而不是再弹出一个新实例。

免费下载 ↓ GitHub ↗

WYSIWYG· 标签页· 文件树· ⌘P· LaTeX· EN / 中文· 6 套主题

01

为多文件写作而生

Typora 有的它都有；下面这些，是 Typora 没有的。

01

标签页

所有文件在同一个窗口里打开。在访达 / 资源管理器中双击文件，是新增一个标签，而不是再启动一个应用。

Ctrl/⌘ Tab
02

文件夹工作区

侧边栏装下整个文件夹：新建、重命名、删除、在系统中显示，全部就地完成；外部改动自动刷新。

Ctrl/⌘ B
03

所见即所得

输入 Markdown，当场渲染。表格、代码高亮、LaTeX 公式、任务列表、斜杠菜单 —— 由 Milkdown / ProseMirror 驱动。

/
04

命令面板

模糊搜索直达任何文件、任何命令。大纲面板点标题即跳转，长文档不再上下翻找。

Ctrl/⌘ P
05

源码模式

一个快捷键切回纯 Markdown 源码；.txt 文件走极速纯文本编辑器，超大文件秒开不卡顿。

Ctrl/⌘ /
06

双语界面

整个界面同时讲 English 和中文。富文本复制保留样式，粘到微信、邮件、Notion 都不丢格式。

EN / 中
02

六种心情，一张书桌

明亮、暗夜，以及四套莫兰迪 —— 雾、鼠尾草、玫瑰、暮色。
